# CS2 Radar Images

Place 1024×1024 PNG radar images here, named after the map (without `de_` prefix):

- `mirage.png` ✅
- `inferno.png` ✅
- `dust2.png`
- `anubis.png`
- `ancient.png`
- `overpass.png`
- `nuke.png`

These are extracted from the CS2 game files at:
`steamapps/common/Counter-Strike Global Offensive/game/csgo/resource/overviews/`

Each map folder has a `*_radar.png` file — copy and rename it.
